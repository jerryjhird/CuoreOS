# CuoreOS
Hobby x86-64 ELF Kernel

## Build Reqiurements ("make")
- GNU Cross Compiler (GCC) (>= 10)
- GNU linker (ld)
- make
- mtools
- git
- dd
- parted

## Requirements ("make run")
- qemu-system-x86_64
- OVMF UEFI firmware
